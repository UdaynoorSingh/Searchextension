Tips & Tricks

  • Instant Search: Select text and press your shortcut, or click the extension icon to search immediately.
  • Keyboard Navigation: While the search input is focused, use the Up and Down arrow keys to cycle through highlighted results.
  • Jump to Result: Click the current iteration index (e.g., 1/5) and type a number to skip directly to it.
  • Typo Tolerance: Use Partial mode or Sounds-alike matching to find text even with spelling mistakes.
  • Advanced Customization: Open the Options page to configure themes, diacritics matching, smooth scrolling, and more.
